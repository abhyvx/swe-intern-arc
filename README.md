# SWE Intern Arc

Live plan for Abhy: C/systems, DSA, Gambit/NetKV, winter intern readiness.

## Use it

Open the GitHub Pages site (or `index.html` locally). Hard-refresh after updates.

Progress saves in this browser (`abhy-plan-v10`). Use **Export / Import** on the Schedule tab to move progress between phone and laptop.

## Daily recalibration

On every open and after each completed step the app:

1. Shows pace status: On track / Behind / Ahead
2. Folds unfinished work **forward in original order** (foundations before later blocks; never skip ahead)
3. Packs remaining steps into realistic daily minute caps through Dec
4. Drops filler drills first if overloaded, keeps core path and deadlines

You cannot open a later skill jump or mark a step complete until earlier blocks in the spine are done.

## Files

- `curriculum.js` – full arc content
- `recalibrate.js` – live schedule engine
- `teach_bank.js` + `enrich.js` – teaching content
- `guides/` – local lesson pages
